# คู่มือการสมัคร Google AdSense สำหรับใช้งานกับ Mystic Cards

การสมัคร **Google AdSense** เพื่อยื่นขอติดโฆษณาในเว็บไซต์แอปไพ่ทาโรต์ของเรา มีขั้นตอนดังต่อไปนี้:

## 📋 สิ่งที่ต้องเตรียมก่อนสมัคร
1. **Google Account (Gmail)** - บัญชีหลักสำหรับรับเงินและจัดการระบบโฆษณา
2. **เว็บไซต์ที่ออนไลน์และเข้าถึงได้จริง (Production URL)** - ต้องไม่ใช่ `localhost` เนื้อหาเว็บต้องพร้อมใช้งานจริง
3. **Domain Name ของตัวเอง** (เช่น `mysticcards.com`) - ทาง AdSense มักจะไม่อนุมัติหากใช้ Subdomain ฟรีในการสมัครครั้งแรก หรือหากเป็น Subdomain ฟรีบางประเภทระบบอาจจะปฏิเสธตั้งแต่ขั้นตอนกรอก URL

---

## 🚀 ขั้นตอนการสมัคร Google AdSense

### 1. เข้าสู่ระบบ AdSense
1. เข้าไปที่เว็บไซต์ [Google AdSense](https://adsense.google.com/start/) 
2. คลิกปุ่ม **"Get started"** (เริ่มต้นใช้งาน)
3. ล็อกอินด้วยบัญชี Gmail ที่ต้องการผูกกับ AdSense

### 2. กรอกข้อมูลเว็บไซต์ของคุณ
- **Your site (เว็บไซต์ของคุณ):** ใส่ URL เว็บไซต์จริงของคุณ (ไม่ต้องมี `https://` หรือ `www.` ก็ได้ เช่น `mysticcards.com`)
  - *หากเว็บไซต์ยังไม่พร้อม สามารถเลือก "I don't have a site yet" (ฉันยังไม่มีเว็บไซต์) ไปก่อนได้*
- **Get more out of AdSense:** เลือกว่าต้องการรับคำแนะนำทางอีเมลหรือไม่
- **Your payment country/territory (ประเทศ/พื้นที่การชำระเงิน):** เลือก **"Thailand"** 
  - ⚠️ **สำคัญ:** ข้อมูลบรรทัดนี้แก้ไขภายหลังไม่ได้ และมีผลกับสกุลเงินรวมถึงขั้นตอนยื่นภาษี
- ทำเครื่องหมายยอมรับข้อตกลงและเงื่อนไข แล้วคลิก **"Start using AdSense"**

### 3. กรอกข้อมูลการชำระเงิน (Payment Information)
เมื่อเข้ามาสู่หน้า Dashboard หลัก (Home) คุณจะพบการตั้งค่า 3 ส่วนหลัก ให้เริ่มที่การชำระเงินก่อน:
1. ไปที่เมนู **Payments > Payment info** (หรือคลิกที่การ์ด "Enter information")
2. **Account type:** เปลี่ยนเป็น "Individual" (บุคคลธรรมดา) หากไม่ใช่รูปแบบบริษัท 
3. กรอก **ชื่อ-นามสกุลจริง** (ภาษาอังกฤษ) ให้ตรงกับบัญชีธนาคารที่คุณวางแผนจะใช้รับเงิน
4. กรอก **ที่อยู่** (Address) ให้ละเอียดและถูกต้อง
  - ⚠️ **สำคัญ:** เมื่อรายได้สะสมของคุณถึง $10 ระบบของ Google จะส่ง "จดหมายรหัส PIN" ที่เป็นกระดาษจริงๆ มาให้ถึงบ้านตามที่อยู่นี้ เพื่อยืนยันว่าคุณมีตัวตนจริง

### 4. เชื่อมต่อเว็บไซต์เข้ากับ AdSense (Connect your site)
1. ไปที่เมนู **Sites** เลือก **Add site** (เพิ่มเว็บไซต์) หรือคลิกจากการ์ดในหน้า Home
2. ระบบจะให้รหัสโค้ด (AdSense snippet script) มา หน้าตาประมาณนี้:
   ```html
   <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-1234567890123456" crossorigin="anonymous"></script>
   ```
3. คัดลอก **Publisher ID** (ส่วนที่ขึ้นต้นด้วย `ca-pub-XXXXXXXXXXXXXXX`) 
4. นำรหัสนั้นไปใส่แทนที่ Placeholder ในโค้ดของแอป:
   - ไฟล์ **`src/app/layout.tsx`** (บรรทัดที่วางแท็ก `<script>`)
   - ไฟล์ **`src/components/AdBanner.tsx`** (ตรง `data-ad-client="..."`)

### 5. รอรับการตรวจสอบ (Review Process)
1. หลังจาก Deploy ฝั่งโค้ดของเราขึ้น Production แล้ว ให้กลับมาที่ AdSense แล้วกด **"Request review"** (ขอรับการตรวจสอบ)
2. บอทของ Google จะเข้าสำรวจโครงสร้างและเนื้อหาเว็บไซต์ว่าถูกแนวทาง (Policy) หรือไม่ ตัวเว็บต้องสามารถเข้าใช้งานได้อย่างมีประสิทธิภาพ, ไม่ใช่ Blank Page, และไม่มีเนื้อหาที่ผิดนโยบายตามของ Google กำหนด
3. กระบวนการนี้อาจใช้เวลาตั้งแต่ **2-3 วัน จนถึง 2-4 สัปดาห์**
  - ในระหว่างนี้ โฆษณาที่อยู่บนเว็บจะไม่ปรากฏขึ้นเลย (หรือเป็นพื้นที่ว่างเปล่า/Placeholder ขึ้นอยู่กับ CSS ของเรา)

### 6. สร้างหน่วยโฆษณา (Ad Units) — หลังผ่านการอนุมัติ
หาก Google ส่งอีเมลแจ้งเตือนว่า *"Good news! Your site is now ready to show AdSense ads."* แล้วให้ดำเนินการดังนี้:
1. เข้าหน้า AdSense Dashboard
2. ไปที่เมนู **Ads > By ad unit** (โฆษณา > ตามหน่วยโฆษณา)
3. เลือกสร้าง **"Display ads"** (โฆษณาแบบดิสเพลย์)
4. ตั้งชื่อให้หน่วยโฆษณานั้นจำง่าย เช่น `"Reading Result Banner"`
5. ตรงตัวเลือก "Ad size" เลือกเป็น **"Responsive"** (ปรับเปลี่ยนตามขนาดหน้าจออัตโนมัติ)
6. คลิกเพื่อบันทึกและสร้างโค้ดโฆษณา
7. จากหน้าข้อมูล Script ที่ได้มา ให้จดรหัส **Ad Slot** (ในแอตทริบิวต์ `data-ad-slot="1234567890"`)
8. นำรหัส Ad Slot นั้นกลับไปใส่ในแอปตรง prop `adSlot` เมื่อเรียกใช้คอมโพเนนต์ `<AdBanner>`
